# شبوق المملكة

موقع تعريفي عربي لشركة متخصصة في الشبوك، المظلات، أعمال الحدادة، والمستودعات.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## إنشاء نسخة النشر

```bash
npm run build
```

سيتم إنشاء ملفات الموقع النهائية داخل مجلد `dist`.

## النشر على GitHub Pages

1. ارفع المشروع إلى مستودع GitHub.
2. من إعدادات المستودع افتح `Pages`.
3. اختر `GitHub Actions` كمصدر النشر.
4. ادفع التغييرات إلى فرع `main`.

ملف النشر داخل `.github/workflows/deploy.yml` سيبني الموقع وينشره تلقائيًا.
